I-C'EST QUOI LE FRONT-END ?

En developpement web, la notion de front-end fait reference à l'ensenble des éléments visible et accéssible directement sur un site web (voir sur une application web ou une application mobile). c'est en d'autre terme la partie visuelle d'un site web affiché par un navigateur. Le HTML, le CSS et le JAVASCRIPT sont les éléments majeurs qu'utilise le front-end en particulier le HTML qui est un élément indispensable à tout site

II-HTML

1-Definition

Le HTML(HperText Markup Language) est un langage de balissage utilisé pour créer voir structurer une page web.Il est utilisé avec d'autre technologie tel que le CSS pour décrire la presentation de la page web et/ou le JAVASCRIPT pour ajouter des fonctionnaluités interactives à la page web.

2-Tim Berners-Lee et le HTML

Le HTML est l'une des trois invention majeurs(avec le HTTP et L'URL) ayant permis l'avènement du web. Crée par Tim Berners-lee, le HTML est une dérivée du SGML(Standard Generalized Markup Language) utilisé par le Conseil Européen pour la Recherche Nucléaire (CERN) . Travaillant pour cette organisation, Berners-lee avait pour mission la création d'une base de données pour les chercheur du CERN.C'est de là que naît son mémo en mars 1989 intitulé "Gestion de l'information" oú sont défini les jalons du HTML.Plus tard il quitte le CERN et Crée le W3C( World Wid Web) d'où le html connaitra une évolution.

3-Du HTML au HTML5

Après la sortie officiel du HTML(Août 1991), ce langage de balises engendre beaucoup de polémique vue qu'il n'est pas formellement défini et cette fonctionnalité est laissé aux differents serveurs.Ainsi, le W3C de Berners-lee conçoit le HTML2.0 compatible à tout sorte de serveurs.Cependant,les utilisateurs se plaignent toujours du fait de ne pas avoir d'influence sur l'apparence de la page web avec le HTML(d'où la création du CSS).De là,il s'ensuit la création du HTML3,2 puis du HTML4 qui croisent encore d'autre imccompatibilités.Du coup en 2003 le W3C met fin au developpement du HTML pour se concentrer sur le XHTML tâché uniquement sur la structuration du document. La dernière version du HTML(HTML5) voit le jours en 2016 suite à quelques bavure entre le W3C et le WHATWG(Web Hypertext Application Technology Working ). Le HTML5 est ainsi un standard en constante évolution. il n'aura pas de HTML6.
